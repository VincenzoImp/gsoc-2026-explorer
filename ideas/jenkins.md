Below you can find project ideas that have been proposed for this year.
New ideas may be proposed by interested mentors or GSoC contributors, such as new features in the core or "write a plugin for MY_TOOL_OR_SERVICE".
Project ideas without potential mentors will be considered, though applicants may need to work with the community and GSoC org admins to find mentors.
To add a new project idea, see:
[
proposing project ideas
](/ideas/jenkins-wp/projects-gsoc-proposing-project-ideas).

The following list contains the project ideas that fully match the Jenkins project idea standard. The scope of these ideas is understood and we don't normally expect deep changes. All ideas have quick-start guidelines and newbie-friendly issues referenced. We welcome contributors to join the mentor teams and invite GSoC contributors to submit project proposal applications related to these ideas.

|
AI Chatbot to Guide User Workflow
|

Develop an AI-powered chatbot integrated into Jenkins to assist users in navigating workflows, accessing documentation, and troubleshooting issues efficiently..

Develop an AI-based chatbot to provide users with quick and intuitive access to Jenkins documentation, plugins, and community resources..

Using alternative tooling with Vite and Antora to build the Jenkins static site and provide versioned Jenkins documentation.

Using a modern and secure method to send Jenkins email notifications via Outlook SMTP with OAuth authentication by adding OAuth support (with client credentials flow) to email-ext..

Build a public, static visualization site that consumes the existing modernization dataset (from GitHub) during its build process to present dashboards and perâplugin reports.

Migrate and redesign the Jenkins.io Contributor Spotlight website using Vite.js and React.js to enhance user experience and modernize the tech stack..

To revamp jenkins.io website's Success Stories feature to update both tooling and UI/UX experience with new design.

To help enhance observability of Jenkins jobs on ci.jenkins.io via the introduction of the use of OpenTelemetry.
