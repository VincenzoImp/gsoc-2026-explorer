## Background

This project will update the data in the package, develop teaching materials and provide a shiny app for wildlife exploration.

## Related work

The [ecotourism](https://vahdatjavad.github.io/ecotourism/) package was developed as part of GSoC 2025. This year the tourism data needs to be completed, because full records were not available in 2025. The [nycflights13](https://nycflights13.tidyverse.org/) and [palmerpenguins](https://allisonhorst.github.io/palmerpenguins/articles/intro.html) are good example data packages.

## Details of your coding project

The project involves

* update data to include most of 2025
* develop teaching materials for instructors of elementary data science and statistics units
* design, implement and deploy a shiny app that can help users learn abour best locations and times to spot wildlife across Australia.

## Expected impact

Data packages have made substantial impact on the community. They are used in classrooms for teaching, and for examples in published research.

## Tests

Contributors, please do one or more of the following tests before contacting the mentors above.

* Easy: Using the template tutorial [here](https://github.com/numbats/ddde/blob/main/week3/tutorial.qmd), write three questions related to the current version of the ecotourism package that will practice joining the occurrence data with tourism data with weather data, for one region, one organism, and one month. 
* Medium: Construct a shiny app that displays a map of Australia, has a menu to select one of the organisms in the current package, and displays the locations of occurrences.
* Hard: Write a function that takes occurrence and weather data and predicts the top five days and times to spot an organism.
